Projets
Application de messagerie instantanée JavaEE - PostGreSQL - Modèle MVC - API REST

CampusTalk est une application web de messagerie instantanée inspirée des réseaux sociaux actuels. L’objectif était de développer une plateforme où les utilisateurs peuvent créer des fils de discussion, y inviter d'autres membres, poster des messages textuels ou des images, et gérer la durée de vie de leurs messages. Développée en JavaEE sur Tomcat avec une architecture MVC, l’application s’appuie sur une base de données relationnelle modélisée via un MCD et un MLD.

Côté front-end, j’ai utilisé Bootstrap pour assurer un design responsive adapté à tous types d’écrans. Le back-end intègre un système d’authentification sécurisé ainsi qu’une protection contre les injections SQL et XSS. J’ai également mis en place un service REST pour exposer certaines données sous forme de JSON. Ce projet m’a permis de consolider mes compétences en développement web fullstack tout en appliquant des bonnes pratiques de structuration et de sécurisation d’une application.

Application d’apprentissage supervisé JavaFX - Analyse - Algorithme k-NN

Dans ce projet, développé en Java avec JavaFX, nous avons implémenté l’algorithme de classification k-Nearest Neighbors (k-NN) afin d’analyser et de classer des données issues de fichiers CSV. L’application permet de charger un jeu de données, de le visualiser sous forme de nuages de points interactifs et de prédire la catégorie d’une nouvelle donnée en temps réel. Conçu en équipe, ce projet a été l’occasion de renforcer notre rigueur en matière de qualité de code et de gestion collaborative. Une attention particulière a été portée à la modularité du code et à la clarté de l’interface utilisateur. Ce projet m’a permis de mieux comprendre les concepts liés à la manipulation et à la visualisation de données, tout en consolidant mes compétences en programmation orientée objet et en gestion de projet.

Exploitation algorithmique d'un graphe JavaFX - Analyse - Algorithme k-NN

Ce projet, réalisé en Java avec JavaFX, consistait à développer une application permettant de calculer le plus court chemin entre deux stations en fonction de différents critères (prix, temps, impact écologique). L'utilisateur peut choisir les modes de transport et privilégier un critère spécifique. Ce projet a renforcé mes compétences en algorithmie sur les graphes, modélisation d'interfaces graphiques, et développement orienté objet.

Manipulation et ventilation de données PostGreSQL

Ce projet consistait à créer et exploiter une base de données à partir d’un fichier CSV lié à la plateforme Parcoursup. J'ai participé à la modélisation des données, la création de la base sous PostgreSQL, ainsi qu’à la ventilation des données pour éviter les redondances.
